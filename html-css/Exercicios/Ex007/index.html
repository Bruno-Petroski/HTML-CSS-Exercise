Texto Marcado

Podemos criar um texto marcado utilizando a tag <mark>
E ele não herda a cor de outros textos marcado

Texto Grande e Pequeno:

Podemos fazer textos grandes utilizando <big>(Obsoleto) e textos pequenos utilizando <small>

Texto Riscado:

Podemos riscar texto (Goiabinha) utilizando <del>

Sublinhar texto:

Podemos sublinhar texto utilizando <ins>
Ou utilizando <u>

Texto Sobrescrito e Texto Subscrito

Sobrescrito serve para escrevermos potencias e coisas do tipo como 102, utilizando <sup>

Subscrito serve para colocar coisas em baixo como H2O, utilizando <sub>
